== Description ==

The most simple WordPress plugin for creating subscriber lists for your blog or small business. You will be able to keep your subscribers in the simple and compact online list with an automated process that is adding all new subscriber emails to your database.
With plugin installation, you are obtaining a simple opt-in form for your blog, landing page, or e-commerce.
Please visit http://madletter.com/ to register and complete plugin installation.

Please send as an email at support@madletter.com to tell us what would you like to add as a new functionality, or if you need help setting up.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/madletter-widget` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to Madletter page.
4. Enter your Madletter API Key (You can get API Key by registering on http://madletter.com and adding domain)
